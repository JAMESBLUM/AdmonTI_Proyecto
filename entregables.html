ENTREGABLES Y RESULTADOS


Los proyectos se llevan a cabo para cumplir objetivos mediante la producción de entregables. Un objetivo se define como una meta hacia la cual se debe dirigir el trabajo, una posición estratégica que se quiere lograr, un fin que se desea alcanzar, un resultado a obtener, un producto a producir o un servicio a prestar. Un entregable se define como cualquier producto, resultado o capacidad único y verificable para ejecutar un servicio que se produce para completar un proceso, una fase o un proyecto. Los entregables pueden ser tangibles o intangibles.



1. Producto único

Un producto único, que puede ser un componente de otro elemento, una mejora o corrección de un elemento o un nuevo elemento final en sí mismo (p.ej., la corrección de un defecto en un elemento final).

2. Un servicio único o la capacidad de realizar un servicio

Un servicio único o la capacidad de realizar un servicio (p.ej., una función de negocio que brinda apoyo a la producción o distribución).


3. Un resultado único

Un resultado único, tal como una conclusión o un documento (p.ej., un proyecto de investigación que desarrolla conocimientos que se pueden emplear para determinar si existe una tendencia o si un nuevo proceso beneficiará a la sociedad).

4. Una combinación única de uno o más productos, servicios o resultados

Una combinación única de uno o más productos, servicios o resultados (p.ej., una aplicación de software, su documentación asociada y servicios de asistencia al usuario).
